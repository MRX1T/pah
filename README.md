# PAH: Python AUR Helper
*The most minimalistic helper for arch user repository by wrote in python*


Installation
=============
1. chmod +x setup.sh
2. ./setup.sh


PAH help
============
*before installation:* python3 pah.py [help, h]  
*after installation:* pah [help, h]
